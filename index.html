Overview

This privacy policy describes how the SproutScout Pinterest Publisher app ("the App") handles data when connecting to the Pinterest API on behalf of Sprout Scout Creations.

Data We Access

The App accesses the following data through the Pinterest API:

  • Pinterest account authentication tokens (OAuth)
  • Board and pin metadata necessary to create and schedule pins

How We Use Data

Data accessed through the Pinterest API is used solely to:

  • Automatically publish product mockup images as Pinterest pins
  • Link pins to Etsy listings for Sprout Scout Creations' digital download shop
  • Drive organic traffic and product discovery

Data Storage

  • Authentication tokens are stored locally on the organization's private infrastructure
  • No user data is stored on third-party servers beyond what Pinterest's own platform retains
  • Data is never sold, shared, or transferred to any third party

Third-Party Services

This app integrates with:

  • Pinterest API — for pin creation and scheduling
  • Etsy — as the destination for listing links

Your Rights

As this is a personal/organizational use app, it is not available to the general public. If you have questions about data handling, contact us at tito.obaisi@gmail.com.

Changes to This Policy
